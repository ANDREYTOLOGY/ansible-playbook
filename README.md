# Ansible Playbook: ClickHouse + Vector + LightHouse

Этот репозиторий содержит Ansible playbook для развертывания стека:

- **ClickHouse** — база данных для хранения логов
- **Vector** — сбор и отправка логов
- **LightHouse** — веб-интерфейс для работы с ClickHouse

## Используемые роли

- ClickHouse — https://github.com/AlexeySetevoi/ansible-clickhouse
- Vector — https://github.com/ANDREYTOLOGY/vector-role
- LightHouse — https://github.com/ANDREYTOLOGY/lighthouse-role
